C.build (url, transport, session_id)
Build the URL to call for a given transport and a given session id, if defined. The returned URL is ensured to be unique, as a timestamp and an incrementing counter is included, as specitifed by the socket.io protocol.

Parameters:

  • url The socket.io URL server
  • transport The transport which will use the URL
  • session_id If defined, the session ID. Else nil.

Returns:

    A string being the url
